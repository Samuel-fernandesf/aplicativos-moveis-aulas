# 2º Bimestre - Aplicações Mobile

Este diretório reúne as atividades desenvolvidas durante o segundo bimestre da disciplina de Aplicativos Móveis, utilizando React Native com Expo.

As atividades têm como foco o aprendizado prático de conceitos fundamentais do desenvolvimento mobile, incluindo componentes, propriedades (props), estados, estilização e manipulação de eventos.

---

## Aulas/Atividades

| Aula | Tema | Descrição |
|------|------|------------|
| [**Aula 5**](./Aula%205/) | Components e Props | Desenvolvimento de novos componentes reutilizáveis, com foco na passagem de Strings, Números e Funções através de props. |

---

## Conteúdos Trabalhados

Durante as atividades deste bimestre foram praticados conceitos como:

- Criação de componentes reutilizáveis
- Organização de projetos React Native
- Passagem de dados via Props
- Passagem de funções entre componentes
- Manipulação de estados com `useState`
- Renderização condicional
- Utilização de `FlatList`
- Eventos com `Pressable`
- Estilização com `StyleSheet`
